# Сервис размещения и аренды жилой недвижимости

Полнофункциональный сервис для размещения и аренды жилой недвижимости с системой подписок.

## Возможности

- 🔐 **Аутентификация пользователей** - регистрация и вход
- 💳 **Система подписок** - три тарифных плана (Базовый, Премиум, Корпоративный)
- 🏠 **Размещение объектов** - владельцы могут размещать свои объекты недвижимости
- 📅 **Бронирование** - арендаторы могут бронировать объекты
- 🔍 **Поиск и фильтрация** - поиск объектов по местоположению, типу, цене
- ⭐ **Рейтинги и отзывы** - система оценки объектов

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Запустите сервер:
```bash
npm start
```

3. Откройте браузер и перейдите по адресу:
```
http://localhost:3000
```

## API Endpoints

### Аутентификация
- `POST /api/auth/register` - Регистрация
- `POST /api/auth/login` - Вход
- `GET /api/auth/me` - Получить текущего пользователя

### Подписки
- `GET /api/subscriptions/plans` - Получить планы подписки
- `GET /api/subscriptions/my` - Получить подписку пользователя
- `POST /api/subscriptions/subscribe` - Создать/обновить подписку

### Недвижимость
- `GET /api/properties` - Получить все объекты
- `GET /api/properties/:id` - Получить объект по ID
- `POST /api/properties` - Создать объект (требует подписку)
- `PUT /api/properties/:id` - Обновить объект
- `DELETE /api/properties/:id` - Удалить объект
- `GET /api/properties/my/listings` - Получить объекты пользователя

### Бронирования
- `POST /api/bookings` - Создать бронирование
- `GET /api/bookings/my` - Получить бронирования пользователя
- `GET /api/bookings/my/properties` - Получить бронирования для объектов пользователя
- `PUT /api/bookings/:id/status` - Обновить статус бронирования

## Тарифные планы

### Базовый ($9.99/месяц)
- Размещение до 3 объектов
- Базовая статистика
- Поддержка по email

### Премиум ($19.99/месяц)
- Неограниченное размещение объектов
- Расширенная статистика
- Приоритетная поддержка
- Продвижение в поиске

### Корпоративный ($49.99/месяц)
- Все возможности Премиум
- API доступ
- Персональный менеджер
- Кастомные интеграции

## Использование

1. **Регистрация/Вход**: Нажмите на иконку профиля в правом верхнем углу
2. **Подписка**: После регистрации выберите тарифный план для размещения объектов
3. **Размещение объекта**: Нажмите "Сдать жильё" и заполните форму
4. **Бронирование**: Нажмите "Забронировать" на карточке объекта

## Технологии

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend**: Node.js, Express
- **База данных**: JSON файлы (для простоты, можно заменить на MongoDB/PostgreSQL)
- **Аутентификация**: JWT токены
- **Хеширование паролей**: bcryptjs

## Примечания

- Для продакшена рекомендуется использовать реальную базу данных (MongoDB, PostgreSQL)
- Добавьте интеграцию с платежными системами (Stripe, PayPal) для реальных платежей
- Настройте переменные окружения для секретных ключей
- Добавьте валидацию на стороне клиента и сервера
- Реализуйте загрузку изображений для объектов
